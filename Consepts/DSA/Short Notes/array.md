## ✅ 1. What is it?

An **Array** is a collection of **same type of elements stored in a single variable**.
👉 Stored in **continuous memory locations**

---

## 🎯 2. Why is it used?

* To store **multiple values in one variable**
* To make data **easy to manage**
* To improve **performance (fast access using index)**

---

## 📍 3. Where is it used?

* In Java programs
* In backend:

  * Handling lists of data
  * API responses

---

## 💻 4. Small code example

```java id="p3k8zx"
public class Test {
    public static void main(String[] args) {

        int[] arr = {10, 20, 30, 40};

        System.out.println(arr[0]); // 10
        System.out.println(arr[2]); // 30
    }
}
```

---

## ⚙️ 5. Internal flow (step-by-step)

1. Array is created
2. Memory allocated (continuous)
3. Elements stored
4. Access using index (0-based)

---

## ⚠️ 6. Common mistake beginners make

* ❌ Index starts from 1 (wrong, starts from 0)
* ❌ Accessing invalid index → error
* ❌ Mixing different data types
