# Stages of Learning Sliding Window

### 1️⃣ Understanding Stage (5 problems)

Goal: understand **when the window expands and shrinks**.

Examples:

* Maximum sum subarray of size k
* Average of subarray of size k
* Minimum size subarray sum
* Longest substring without repeating characters
* Longest substring with k distinct characters

---

### 2️⃣ Pattern Recognition Stage (10 problems)

Now you should start seeing:

```text
array / string
+
subarray / substring
+
contiguous elements
```

Common problems:

* Longest repeating character replacement
* Permutation in string
* Minimum window substring
* Fruits into baskets
* Subarray product less than k

---

### 3️⃣ Mastery Stage (10 problems)

Now variations become easier.

Examples:

* Count number of nice subarrays
* Binary subarrays with sum
* Longest subarray with sum k
* Maximum consecutive ones III
* Sliding window maximum

---

# How to Recognize Sliding Window Pattern

Look for these clues:

```text
array or string
+
contiguous subarray / substring
+
maximize or minimize something
```

Example keywords:

* **longest**
* **shortest**
* **maximum**
* **minimum**
* **subarray**
* **substring**

These often mean **sliding window**.
