## ✅ 1. What is it?

A **Data Type** defines:
👉 *“What kind of data a variable can store.”*

Example: number, text, true/false

---

## 🎯 2. Why is it used?

* To **store data correctly**
* To control **memory usage**
* To avoid **errors**

---

## 📍 3. Where is it used?

* In **variables**
* In Spring:

  * DTO fields
  * Entity fields
  * API request/response

---

## 💻 4. Small code example

```java id="h3k9zw"
public class Test {
    public static void main(String[] args) {

        int age = 22;          // integer
        double salary = 50000; // decimal
        char grade = 'A';      
        boolean isActive = true;
        String name = "Rahul";
    }
}
```

---

## ⚙️ 5. Internal flow (step-by-step)

1. Variable is declared
2. Data type decides memory
3. Value is stored
4. JVM handles operations based on type

---

## ⚠️ 6. Common mistake beginners make

* ❌ Using wrong data type (int vs double)
* ❌ Confusing `char` and `String`
* ❌ Not initializing variables
