# 📝 Notes Management System

A beautiful, feature-rich Notes Management System built with React.js. Perfect for organizing AI-generated notes (ChatGPT, Claude, etc.) with markdown support!

## ✨ Features

### 📁 Smart Organization
- **Subject → Topic → Notes** hierarchy
- Auto-detects folder structure
- 200+ notes organized perfectly

### 🎨 40+ Beautiful Themes
- Dark, Light, Ocean, Forest, Galaxy
- VS Code, GitHub, Tailwind inspired
- **Casino Royale** (Gold theme with animations)
- Las Vegas, Poker Night, Blackjack themes
- Live theme switching with persistence

### 🔍 Powerful Search
- Real-time search across all notes
- Search term highlighting in content
- Auto-expand matching topics
- Keyboard shortcut: `Ctrl+K`

### 📖 Markdown Support
- Full markdown rendering
- Syntax highlighting for code blocks
- Tables, blockquotes, lists
- Header anchors for easy navigation

### 💾 Smart Features
- Session persistence (remembers last note)
- Favorites/Bookmarks system
- Recent notes tracking
- One-click copy note
- Fullscreen reading mode
- Word count display

## 🛠️ Tech Stack

- **React.js** - Frontend framework
- **React Markdown** - Markdown rendering
- **React Syntax Highlighter** - Code highlighting
- **React Icons** - Beautiful icons
- **CSS Variables** - Dynamic theming
- **LocalStorage** - Data persistence
